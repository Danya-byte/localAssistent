# Branding Assets

Generated liquid-glass asset pack for the Windows release pipeline.

- `app.ico`: main executable icon
- `installer.ico`: installer icon
- `wizard-back.png`: wide wizard background
- `wizard-side.png`: side illustration for the installer
- `wizard-small.png`: small accent image for the wizard header

These files are used by `scripts/build.ps1` and `scripts/installer.iss`.
